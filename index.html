Goals

Frontend

  • Master CSS Grid and Flexbox for creating complex, responsive layouts.
  • Gain proficiency in hooks, context API, and advanced state management with Redux or MobX. Create custom hooks and higher-order components with react.
  • Learn and use utility-first CSS frameworks like Tailwind CSS for rapid UI development.
  • Learn the differences between rebasing and merging and when to use each.
  • Engage in pull requests and code reviews to enhance code quality and team collaboration.

Backend

  • Learn to consume and integrate RESTful APIs and GraphQL for data fetching and manipulation.
  • Learn to implement authentication mechanisms like JWT, OAuth, and session management in your applications.
  • Explore SSR with frameworks like Next.js for React or Nuxt.js for Vue to improve performance and SEO.
  • Learn Docker for containerizing applications and Kubernetes for orchestrating them.
  • ELearn and implement web security best practices such as Content Security Policy (CSP), Cross-Site Scripting (XSS) prevention, and Secure Sockets Layer (SSL) to protect applications from vulnerabilities.

Web3

  • Understand the architecture of decentralized applications, including the interaction between the frontend, smart contracts, and blockchain.
  • Learn to connect a DApp to a blockchain using libraries like Web3.js or ethers.js.
  • Learn about major DeFi protocols such as Uniswap, Aave, Compound, and their underlying mechanisms.
  • Understand concepts like liquidity pools, yield farming, staking, and decentralized exchanges (DEXs).
  • Learn how to create and participate in DAOs using platforms like Aragon or DAOstack.
